## What maintainers can access

Maintainers hold administrative access to the repository, which includes:

- merging pull requests into the `master` branch;
- publishing releases to crates.io, pub.dev, Maven Central, the Swift Package
  Manager distribution, and GitHub Releases;
- managing repository settings, branch protection, and CI/CD secrets.

All maintainers are required to protect their GitHub accounts with multi-factor
authentication.
