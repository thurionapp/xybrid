title: 快速开始
description: 几分钟内开始使用 Xybrid
---

只需几行代码即可在您的应用中运行 Xybrid。选择下方的 SDK 开始。

<Tabs items={['Flutter', 'Swift', 'Kotlin', 'Unity', 'Rust', 'CLI']}>
  <Tab value="Flutter">
**安装**

```yaml
# pubspec.yaml
dependencies:
  xybrid_flutter: ^0.1.2
```

```bash
flutter pub get
```

**运行推理**

```dart
import 'package:xybrid_flutter/xybrid_flutter.dart';

// 默认即可本地运行。在 dashboard.xybrid.dev 免费获取密钥即可查看
// 推理轨迹：Xybrid.init(apiKey: '...')
await Xybrid.init();

// 加载模型并运行文本转语音
final model = await Xybrid.model('kokoro-82m').load();
final result = await model.run(
  XybridEnvelope.text("Hello from Xybrid!"),
);

// result.audioBytes 包含生成的语音音频
```

**预期输出：** 包含合成语音的音频字节。
  </Tab>
  <Tab value="Swift">
**安装**

在 Xcode 中通过 Swift Package Manager 添加：

**File > Add Package Dependencies**，输入：

```
https://github.com/xybrid-ai/xybrid
```

或添加到 `Package.swift`：

```swift
dependencies: [
    .package(url: "https://github.com/xybrid-ai/xybrid", from: "0.1.2")
]
```

**运行推理**

```swift
import Xybrid

// 默认即可本地运行。在 dashboard.xybrid.dev 免费获取密钥即可查看
// 推理轨迹：Xybrid.initialize(apiKey: "...")
Xybrid.initialize()

// 从注册表加载模型
let loader = XybridModelLoader.fromRegistry(modelId: "kokoro-82m")
let model = try loader.load()

// 运行文本转语音
let envelope = XybridEnvelope.text(text: "Hello from Xybrid!")
let result = try model.run(envelope: envelope)

// result.audioBytes 包含生成的语音音频
```

**预期输出：** `result.success == true`，音频字节在 `result.audioBytes` 中。
  </Tab>
  <Tab value="Kotlin">
**安装**

```kotlin
// build.gradle.kts
dependencies {
    implementation("ai.xybrid:xybrid-kotlin:0.1.2")
}
```

**运行推理**

```kotlin
import ai.xybrid.XybridModelLoader
import ai.xybrid.XybridEnvelope

// 默认即可本地运行。在 dashboard.xybrid.dev 免费获取密钥即可查看
// 推理轨迹：Xybrid.init(context, apiKey = "...")
Xybrid.init(context)

// 从注册表加载模型
val loader = XybridModelLoader.fromRegistry("kokoro-82m")
val model = loader.load()

// 运行文本转语音
val envelope = XybridEnvelope.Text(text = "Hello from Xybrid!")
val result = model.run(envelope)

// result.audioBytes 包含生成的语音音频
```

**预期输出：** `result.success == true`，音频字节在 `result.audioBytes` 中。
  </Tab>
  <Tab value="Unity">
**安装**

在 Unity 中，前往 **Window > Package Manager > + > Add package from git URL**，输入：

```
https://github.com/xybrid-ai/xybrid.git#upm
```

`#upm` 始终指向最新发布的 Unity 构建。如需固定版本，请使用[仓库标签页](https://github.com/xybrid-ai/xybrid/tags)中的 `upm/*` 标签。

**运行推理**

```csharp
using Xybrid;
using UnityEngine;

// 默认即可本地运行。在 dashboard.xybrid.dev 免费获取密钥即可查看
// 推理轨迹：XybridClient.Initialize(apiKey: "...")
XybridClient.Initialize();

// 加载 TTS 模型并生成 NPC 对话
using var model = XybridClient.LoadModel("kokoro-82m");
using var result = model.Run(Envelope.Text("Welcome, traveler. The road ahead is dangerous."));

result.ThrowIfFailed();
Debug.Log($"输出: {result.Text}");
Debug.Log($"延迟: {result.LatencyMs}ms");
```

**预期输出：** `result.Success == true`，包含来自 TTS 模型的音频输出。
  </Tab>
  <Tab value="Rust">
**安装**

```toml
# Cargo.toml
[dependencies]
xybrid = "0.1.2"
```

**运行推理**

```rust
use xybrid::ModelLoader;
use xybrid::ir::{Envelope, EnvelopeKind};

fn main() -> Result<(), Box<dyn std::error::Error>> {
    // 从注册表加载模型
    let model = ModelLoader::from_registry("kokoro-82m").load()?;

    // 运行文本转语音
    let input = Envelope::new(EnvelopeKind::Text("Hello from Xybrid!".into()));
    let result = model.run(&input, None)?;

    // result 包含生成的语音音频
    Ok(())
}
```

LLM 模型需要 llama.cpp 后端 — 通过 `features = ["llm-llamacpp"]` 启用。

**预期输出：** 包含合成语音音频的结果 Envelope。
  </Tab>
  <Tab value="CLI">
**安装**

```bash
# macOS / Linux
curl -sSL https://raw.githubusercontent.com/xybrid-ai/xybrid/master/install.sh | sh
```

```powershell
# Windows (PowerShell)
irm https://raw.githubusercontent.com/xybrid-ai/xybrid/master/install.ps1 | iex
```

预编译二进制与源码构建方式参见[命令行指南](/docs/cli)。

**运行推理**

```bash
# 列出可用模型
xybrid models list

# 运行文本转语音
xybrid run --model kokoro-82m --input-text "Hello from Xybrid!" --output hello.wav

# 运行语音转文字
xybrid run --model whisper-tiny --input-audio recording.wav
```

**预期输出：**

```
Transcription: "Hello, how can I help you?"
```
